# VNNBIZS - ระบบลงประกาศบริษัท

ระบบลงประกาศเชิญประชุมผู้ถือหุ้นประจำปีสำหรับบริษัทตามกฎหมายไทย

## คุณสมบัติหลัก

### สำหรับผู้ใช้ทั่วไป:

- ✅ สมัครสมาชิกและเข้าสู่ระบบ
- ✅ เติมเครดิต (Wallet) ผ่านการโอนเงินธนาคาร
- ✅ สร้างประกาศเชิญประชุมผู้ถือหุ้น
- ✅ อัพโหลด PDF และรูปภาพประกอบ
- ✅ ดูประวัติการลงประกาศ
- ✅ ตรวจสอบเครดิตคงเหลือ

### สำหรับแอดมิน:

- ✅ อนุมัติ/ปฏิเสธการเติมเครดิต
- ✅ ตรวจสอบสลิปการโอนเงิน
- ✅ ดูสถิติการใช้งานระบบ

## ฐานข้อมูล

### ตารางสำคัญ:

- **online_placard_users** - ข้อมูลผู้ใช้
- **placard** - ประกาศบริษัท
- **credit_coin** - แพ็คเกจเครดิต
- **credit_transactions** - ประวัติการเติมเครดิต

## แพ็คเกจเครดิต

| แพ็คเกจ         | ราคา   | เหรียญ    |
| ---------------------- | ---------- | --------------- |
| หนึ่งครั้ง   | 50 บาท  | 1 เหรียญ  |
| หกครั้ง         | 300 บาท | 6 เหรียญ  |
| สิบสองครั้ง | 600 บาท | 12 เหรียญ |

## การติดตั้ง

1. อัพโหลดไฟล์ทั้งหมดไปยัง `/website/www.vnnbizs.com/new/`
2. สร้างโฟลเดอร์สำหรับอัพโหลดไฟล์:

```bash
mkdir -p uploads/slips uploads/pdfs uploads/images
chmod 777 uploads/slips uploads/pdfs uploads/images
```

3. ตรวจสอบการตั้งค่าฐานข้อมูลใน `config/db-config.php`
4. ตั้งค่า Nginx (ทำแล้ว):

   - รองรับทั้ง www.vnnbizs.com และ vnnbizs.com
   - SSL/HTTPS เปิดใช้งาน
   - Client max body size: 2GB

## การใช้งาน

### สำหรับผู้ใช้:

1. เข้าไป https://www.vnnbizs.com/new/
2. คลิก "เข้าสู่ระบบ / สมัครสมาชิก"
3. สมัครสมาชิกใหม่หรือเข้าสู่ระบบ
4. เติมเครดิตโดยโอนเงินและอัพโหลดสลิป
5. รอแอดมินอนุมัติ (ภายใน 24 ชม.)
6. สร้างประกาศเชิญประชุมผู้ถือหุ้น

### สำหรับแอดมิน:

1. เข้าสู่ระบบด้วยบัญชีแอดมิน (groupid = 1)
2. คลิกปุ่ม "Admin" ในเมนูบน
3. ตรวจสอบสลิปและอนุมัติ/ปฏิเสธคำขอ
